## DOM API
1. `요소1.appendChild(요소2);` 요소2를 요소1 자식으로 붙이기
2. `요소.innerHTML = '';` 하위 내용 싹 지우기
3. `요소.innerHTML = '문자열';` 오버라이트, 1번과 비교해서 기억
